## 🚀 Projects

### 📍 Mapier.ai
Location-based social media platform for geo-tagged content discovery. Automated multi-source content ingestion pipelines, post creation workflows, and group-based permission systems.

`React Native` `FastAPI` `PostgreSQL` `Python`

---

### ❤️ Corazón
Platform with 860+ resources for Latinx communities in Chicago. Features an AI voice assistant (Groq + ElevenLabs), automated scraping pipelines, and scalable FastAPI endpoints.

`FastAPI` `React` `PostgreSQL` `ElevenLabs` `Groq`

---

### 🗺 MyYapa
Web platform mapping student discounts across Chicago — aggregating location-based opportunities into a centralized discovery system with automated data pipelines.

`React Native` `TypeScript` `Google Maps API` `OAuth` `PostgreSQL`

---

### 🪐 NASA SpaceApps — Exoplanet Classifier
ML classification system for exoplanet detection using XGBoost, CatBoost, and Random Forest. Applied SHAP-based feature attribution with an interactive web interface for model visualization.

`XGBoost` `CatBoost` `SHAP` `React`
